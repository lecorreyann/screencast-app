# ScreenCast

Partage d'écran web ultra-simple. Aucune installation.

## Déploiement

### Railway / Render / Fly.io (recommandé)

1. Push le code sur GitHub
2. Connecte ton repo sur Railway/Render/Fly
3. Deploy

### VPS / Serveur

```bash
npm install
npm start
```

Le serveur écoute sur le port `3000` (ou `$PORT`).

## Utilisation

1. L'utilisateur va sur `https://screencast-app.onrender.com/`
2. Clique **Partager mon écran**
3. Sélectionne son écran
4. Copie le lien et l'envoie
5. Le spectateur ouvre le lien → voit l'écran

C'est tout. Rien à installer.
